# Memory Maker User App

Public release Flutter source for the Memory Maker user app.

## App name
Memory Maker

## Codemagic environment variables
- SUPABASE_URL
- SUPABASE_ANON_KEY
- APP_URL
- DEMO_MODE

Do not put SUPABASE_SERVICE_ROLE_KEY inside the mobile app.

## Build
Use Codemagic YAML workflow: android-apk.
